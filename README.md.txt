# Netflix Content Analysis

## Objective
Perform Exploratory Data Analysis on Netflix Movies and TV Shows dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data Cleaning
- Missing Value Treatment
- Exploratory Data Analysis
- Data Visualization
- Business Insights

## Visualizations
1. Movies vs TV Shows
2. Top Countries
3. Release Trends
4. Rating Distribution
5. Release Year Histogram
6. Missing Value Heatmap

## Outcome
Generated actionable insights regarding Netflix content trends and distribution.
